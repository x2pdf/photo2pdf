## Tips
1.在MacOS首次安装打开之后，通常需要用户在Mac的 "设置-安全与隐私" 里面允许本应用运行，否则系统可能会提示本应用文件破损，不能打开运行之类的信息。

2.通常情况是使用默认的设定即可，如需要改动设定，请阅读文档信息之后再正确设定。
